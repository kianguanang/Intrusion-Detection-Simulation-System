Requirements

  This program is built using java version 19.0.1

Compilation instruction

  To compile the program, use:

    javac IDS.java

  To run the program, use:
    
    java IDS <event filename> <stats filename> <number of days>
    Example: java IDS Events.txt Stats.txt 20

It is recommended to use a number greater than 10 for number of days for anomaly detection
